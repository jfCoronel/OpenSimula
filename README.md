# OpenSimula
Component-based time simulation environment in Python

## Main Developers

* [Juan F. Coronel](http://jfc.us.es), Universidad de Sevilla

## Package dependencies

- NumPy (v > 1.25)
- Pandas (v > 1.5)
- Scipy  (v > 1.10)

Dependencies for development:

- ipykernel
- plotly
- pytest
- mkdocs
- build
- twine
